1：Python基础教程

Jul 18, 2017 | 阅读

Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。
Python 由Guido van R0ssum 于1989年底发明，第一个公开发行版发行与1991年。像Perl(不懂可以百度哦) 语言一样，Python源代码同样遵循GPL(GNU General Public License)协议。

Python被广泛应用于后端开发、游戏开发、网站开发、科学运算、大数据分析、云计算，图形开发等领域。

Python在软件质量控制、提升开发效率、可移植性、组件集成、丰富库支持等各个方面均处于先进地位。

python具有 简单、易学、免费、开源、可移植、可扩展、可嵌入、面向对象 等 优点，它的面向对象甚至比java和C#.net更彻底

本教程主要针对Python3.x版本的学习。

Python 3.0版本，常被称为Python 3000 ，或简称Py3K。相对于Python早期的版本，这是一个较大的升级。为了不带入过多的累赘，Python3.0在设计的时候没有考虑向下兼容。

谁适合阅读本教程？

本教程适合想从零开始学习Python编程语言的开发人员。当然本教程也会对一些模块进行深入，让你更好的了解Python的应用。
